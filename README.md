# PerturbSeqEngine

Perturb-seq analysis engine for CRISPR perturbation effect quantification, gene program discovery, and causal gene identification.

## Features

- CRISPR perturbation transcriptional effect scoring
- NMF-based gene program discovery from perturbation responses
- Essential gene identification from viability screens
- Perturbation specificity and on-target effect quantification
- Causal gene network inference from perturbation data

## Results

5000 cells, 100 perturbations; Essential=20%; 8 NMF programs; Specificity=0.990

## Usage

```bash
pip install numpy scipy matplotlib
python perturb_seq_engine.py
```
